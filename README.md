## How to Use
1. Click `Use this template`, or fork the repo 
2. Update [vars.sh](./vars.sh)
3. Review the [init](./init) scripts, enable/disable parts
4. Run `./init.sh`

## Troubleshooting
All scripts are idempotent - hack & rerun until it works.
